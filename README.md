## âœï¸ Customization

### Update Your Info

Open `index.html` and edit:

**Personal Details:**
- Line ~270: Update hero title and tagline
- Line ~290: Edit about section text
- Line ~315: Modify skills/tech stack

**Projects:**
- Line ~350+: Update project cards with your actual projects
- Replace placeholder links with your live demos and GitHub repos

**Contact:**
- Line ~630+: Add your actual email and social media links

**Colors (Optional):**
- Line ~45-55: Modify CSS variables for custom colors

### Add Your Own Projects

Replace the placeholder projects with your real ones:

```html
<div class="card">
    <div class="relative z-10">
        <h3 class="text-2xl font-bold mb-3">Your Project Name</h3>

        </p>
```
